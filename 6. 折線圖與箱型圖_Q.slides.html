練習¶

  • 請依據Released_Year及Certificate繪製平均分數折線圖
    • Released_Year取大於2014年的資料
    • Certificate取大於U、A、UA三類的資料
    • 取Released_Year及Certificate兩者的平均分數（平均分數取到小數點第一位）
    • 請設定x、y軸名稱、圖片title並呈現圖例
    • 資料點A以圓形呈現、U以三角形呈現、UA以方形呈現
    • 資料線A以實線呈現、U以虛線呈現、UA以點線呈現

## 範例 - 堆疊折線圖基礎
import matplotlib.pyplot as plt
import calendar

plt.figure(figsize=(10, 5))
month = range(1, 13)
working_time = [6, 2, 7, 1, 7, 9, 8, 6, 8, 9, 6, 8]
rest_time = [8, 4, 5, 3, 6, 9, 5, 3, 8, 3, 2, 9]
plt.stackplot(month, working_time, rest_time, labels=["Work", "Rest"])
plt.axis([1, 12, 0, 20])
plt.xticks(range(1, 13), calendar.month_abbr[1:])
plt.legend(fontsize=16)
plt.show()

import matplotlib.pyplot as plt  #引用

plt.boxplot(data, labels, sym, patch_artist)   #繪圖

## 範例 sym, patch_artist
import random
import matplotlib.pyplot as plt

data = []
data1 = [random.randint(1, 100) for i in range(50)]
data1.append(200)
data2 = [random.randint(100, 200) for i in range(51)]
data.append(data1)
data.append(data2)
plt.figure(figsize=(10, 5))
plt.subplot(121)
plt.boxplot(data, labels=["one", "two"], sym='x')
plt.subplot(122)
plt.boxplot(data, labels=["one", "two"], patch_artist=True)
plt.show()

異常值標記設定¶

flierprops({markeredgecolor:???, markerfacecolor:???, marker:???, markersize:???})

# markeredgecolor: 標記邊線顏色
# markerfacecolor: 標記顏色
# marker: 標記樣式
# markersize: 標記大小

## 範例 - flierprop

import random
import matplotlib.pyplot as plt

data = []
data1 = [random.randint(1, 100) for i in range(50)]
data1.append(200)
data2 = [random.randint(100, 200) for i in range(51)]
data.append(data1)
data.append(data2)
mark_setting = {'markeredgecolor': 'red', 'markerfacecolor': 'white', 'marker': 'X', 'markersize': 20}
plt.figure(figsize=(10, 5))
plt.boxplot(data, labels=["one", "two"], flierprops=mark_setting)
plt.show()

## 範例 - 水平箱型圖

import random
import matplotlib.pyplot as plt

data = []
data1 = [random.randint(1, 100) for i in range(50)]
data1.append(200)
data2 = [random.randint(100, 200) for i in range(51)]
data.append(data1)
data.append(data2)
mark_setting = {'markeredgecolor': 'red', 'markerfacecolor': 'white', 'marker': 'X', 'markersize': 20}
plt.figure(figsize=(10, 5))
plt.boxplot(data, labels=["one", "two"], flierprops=mark_setting, vert=False)
plt.show()
